# PSEUDOCODE Canon Guard

## Rule

`ajax-kernel/PSEUDOCODE_MAP/` is the sole live canonical and exportable pseudocode layer.

## Deprecated Source

`AJAX_HOME/PSEUDOCODE_MAP/` is preserved as a deprecated source archive.
It may inform future audits, but it does not define live truth by default.

## Review Check

If a change:

- edits ROOT pseudocode as if it were live canon,
- proposes automatic sync from ROOT into KERNEL,
- or leaves both trees looking simultaneously authoritative,

then stop and treat that change as drift until an explicit audit says otherwise.
